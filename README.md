# How to
- Download NLTK(Natural Language ToolKit) on your PC
- Import ea.json in your MongoDB local server
- Navigate to Flask/
- Run zxc.py using CMD/Bash
- Go to http://127.0.0.1:5000
- Enter your query
- Hit Return
